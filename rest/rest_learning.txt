# REST

	Definição
		- REST (REpresentational State Transfer)
		- comunicação heterogênea de aplicação-para-aplicação;
		- um paradigma arquitetônico;
		
		
	
	
	RESTful
		- é um serviço web que usufrui deste paradigma;
